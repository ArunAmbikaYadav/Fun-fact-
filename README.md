# Fun Facts App

Welcome to the Fun Facts App! This simple single-page application displays random fun facts at the click of a button. It's built using HTML, CSS, and [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript), and is perfect for learning or teaching basic web development concepts.
